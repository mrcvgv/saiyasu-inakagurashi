# クローラー仕様書 — 掘り出し物件ラボ

## 3層アーキテクチャ

1. **発見** — どのページを巡回すべきか探す
2. **取得** — HTML/JSON/PDF/動的ページから内容を取る
3. **正規化** — 住所・価格・賃料・面積等を統一

---

## 対象ソース分類

### A. 自治体系
- 空き家バンク
- 移住支援ページ
- 補助金・助成金ページ
- 空き家改修補助 / 家賃補助 / 空き家利活用制度
- 地域おこし協力隊 / 定住支援ページ

### B. 民間不動産系
- 賃貸ポータル / 売買ポータル / 土地専門系
- 事業用物件 / 古民家・別荘系サイト

### C. 二次情報系
- 自治体まとめサイト / 空き家紹介メディア / 地方移住情報メディア

### D. 独自登録系
- ユーザー投稿 / 不動産会社掲載 / 自治体担当者からの情報提供

## OpenClaw取得方式

### web_fetch（HTTP取得）
- 静的HTMLページ / 一覧ページ / 詳細ページ
- 制度説明ページ / 補助金概要ページ

### web_search（検索発見）
- 「○○市 空き家バンク」
- 「○○町 移住支援 補助金」
- 「○○県 賃貸 格安」

### Browser tool（ブラウザ自動化）
- JSレンダリング前提の物件一覧
- ページネーションがボタン式 / lazy load
- 検索フォーム入力 / 地図UI依存 / ログイン後画面
- PDFダウンロード導線がJS制御 / SPA系サイト

## 挙動一覧

### 1. ソース発見
- 全国自治体一覧 → 都道府県→市区町村単位で巡回
- 候補語: 空き家バンク / 移住支援 / 補助金 / 空き家改修 / 定住支援 / 家賃補助 / 土地売却 / 公有財産売却
- 自治体公式ドメイン優先判定
- 民間サイトは検索条件URL定義 / 地域別一覧URL収集

### 2. 一覧巡回
- 一覧ページ取得 / 次ページ判定 / 全件取得上限
- 重複URL排除 / 掲載終了・成約済み除外
- 一覧カードから暫定情報取得（タイトル/価格/所在地/面積/間取り/画像/更新日/種別）

### 3. 詳細ページ抽出

#### 不動産共通
物件名 / URL / 物件ID / 価格 / 賃料 / 管理費 / 敷金・礼金 / 所在地 / 緯度経度用住所 / 交通 / 土地面積 / 建物面積 / 間取り / 築年 / 構造 / 用途地域 / 建ぺい率・容積率 / 接道 / 現況 / 引渡時期 / 取引態様 / 写真 / コメント / 取扱会社

#### 賃貸専用
月額賃料 / 初期費用概算 / 更新料 / 保証会社必須有無 / 駐車場 / ペット可 / DIY可 / 事務所利用可 / 短期不可 / 高齢者可 / 外国籍可

#### 土地専用
坪単価 / 上下水道 / ガス / 建築条件 / 農地転用必要有無 / 再建築可否 / 傾斜地・崖条例リスク / 市街化区域・調整区域 / 地目

#### 空き家バンク専用
自治体名 / 掲載番号 / 売買・賃貸区分 / 改修必要度 / 補助金対象可否 / 登録日 / 問い合わせ先 / 内覧条件 / 地域条件 / 利用条件 / 自治会参加条件

### 4. 補助金・制度抽出
自治体名 / 制度名 / 制度URL / カテゴリ（移住支援/空き家改修/住宅取得/家賃補助/子育て支援/起業支援） / 支給額 / 補助率 / 対象者条件 / 年齢条件 / 世帯条件 / 転入条件 / 就業条件 / 申請期限 / 予算上限 / 併用可否 / 担当課 / 電話番号

特殊処理: PDF取得 / 年度更新検知 / 失効制度アーカイブ / 旧制度・新制度区別

### 5. PDF対応
PDFリンク抽出 / ダウンロード / 制度系か物件チラシか判定 / OCR必要判定 / テキスト埋め込みPDF直接抽出 / 表形式項目マッピング / 更新年度抽出

### 6. 正規化
- 価格: 0円/応相談/50万円/1,200万円/月3.8万円 → 数値化
- 面積: ㎡/坪 → 統一、建物/土地分離
- 住所: 都道府県/市区町村/町名/丁目番地/非公開フラグ
- 種別: 土地/中古戸建/空き家/古民家/アパート/店舗付住宅/倉庫/農地付き
- 取引区分: 売買/賃貸/両対応

### 7. 重複排除
URL一致 / 物件ID一致 / 住所＋価格＋面積＋画像類似 / 自治体版と民間版の紐づけ / 人間確認キュー

### 8. 更新検知
新規掲載 / 価格変更 / 賃料変更 / 状態変更（募集中→成約） / 写真変更 / 制度金額変更 / 制度終了 / 差分のみ通知

### 9. フィルタリング

#### 基本フィルタ
売買/賃貸 / 土地/建物/空き家 / 価格帯/家賃帯 / エリア / 土地面積/建物面積 / DIY可 / 駐車場 / 店舗可 / 倉庫付き / 農地付き / 補助金あり

#### "掘り出し" フィルタ
相場より安い / 0円・100万以下・月1万以下 / 写真少ない穴場 / 自治体直掲載 / 補助金合算で実質負担小 / DIY再生余地大

### 10. スコアリング
価格インパクト / 広さ÷価格比 / 補助金込み実質負担 / DIYポテンシャル / 住みやすさ / 事業向き / "掘り出し度"

### 11. 通知
新着物件 / 新着補助金 / 価格下落 / 指定エリア新規 / 指定家賃帯新規 / 条件マッチ / Discord / LINE / メール / Slack

### 12. エラー耐性
タイムアウトリトライ / 403・429バックオフ / CAPTCHA検知 / HTML構造変更検知 / パーサ破損アラート / PDF取得失敗再実行 / JSレンダリング失敗→Browserフォールバック / Browser失敗→スナップショット保存

### 13. 法務・運用安全
robots.txt確認 / 利用規約確認 / 過度アクセス回避 / クロール間隔制御 / User-Agent管理 / 取得元URL保持 / 元ページリンク維持 / 問い合わせ先正確保持 / 個人情報保存制限 / ログ保存
