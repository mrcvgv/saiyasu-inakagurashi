# SAIYASU_VISION.md — 最安田舎暮らし ビジョン

## サイト名

**最安田舎暮らし** (saiyasu-inakagurashi)

ブランド名変更予定: **掘り出し物件ラボ**

## コンセプト

「日本全国の格安物件を、誰でも簡単に探せるサイト」

- 空き家バンク、格安不動産ポータル、競売・公売を横断検索
- 0円物件 / 100万円以下 / 格安賃貸 / DIYリノベ向け を特集
- 移住補助金情報とセットで表示
- SEO重視で検索流入を最大化

## ターゲットユーザー

1. **地方移住を検討している都市部在住者** (30-50代)
2. **格安で土地・建物を探している個人投資家**
3. **DIYリノベに興味がある層**
4. **田舎で安く暮らしたいセミリタイア層**
5. **週末移住・二拠点生活を検討中の人**

## 差別化ポイント

| 既存サイト | 最安田舎暮らしの強み |
|---|---|
| 空き家バンクは市町村ごとにバラバラ | 全国横断で検索できる |
| SUUMO/HOME'Sは高価格帯が中心 | 0円〜100万円の格安帯に特化 |
| 補助金情報が別サイトに分散 | 物件と補助金をセットで表示 |
| 競売・公売は専門サイトのみ | すべて統合して横断検索 |

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS v4
- **DB**: Supabase (PostgreSQL)
- **ホスティング**: Vercel
- **データ源**: 空き家ハンター (別プロジェクト) → Supabase

## このプロジェクトがやらないこと

- スクレイピング (空き家ハンターの責務)
- OpenClaw依存の処理
- 物件データの直接収集
- データスキーマの勝手な変更
