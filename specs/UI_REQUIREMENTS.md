# UI_REQUIREMENTS.md — UI/デザイン要件

## デザイン方針

- **シンプル・クリーン・信頼感** — 不動産サイトとしての信頼性
- **モバイルファースト** — ターゲットの多くがスマホで検索
- **表示速度優先** — SSR + 軽量CSS (Tailwind)
- **SEOフレンドリー** — セマンティックHTML, 適切なheading構造

## カラースキーム

| 用途 | 色 | Tailwind |
|---|---|---|
| メインカラー | グリーン | green-700 / green-900 |
| アクセント | ブルー | blue-600 / blue-800 |
| 背景 | 白 / 薄グレー | white / gray-50 |
| テキスト | 濃グレー | gray-900 / gray-700 |
| 価格強調 | レッド / オレンジ | red-600 / orange-700 |

## レイアウト

- ヘッダー: ロゴ + ナビゲーション (検索, 物件一覧, 補助金, ガイド, 相談)
- メイン: max-w-5xl, px-4, py-6
- フッター: サイト情報 + リンク集

## コンポーネント仕様

### ListingCard

- 画像 (左 or 上)
- タイトル (太字)
- 価格 (大きめ, 色付き)
- 所在地 (県 + 市)
- タグ (小さめ pill)
- 面積情報 (あれば)
- リンク → /listings/[id]

### SearchForm

- 都道府県セレクト
- 価格帯セレクト (0円, 100万以下, 300万以下, 500万以下, 1000万以下)
- キーワード入力 (任意)
- 検索ボタン

### SubsidyCard

- 制度名 (太字)
- 都道府県 / 市区町村
- 金額
- 条件 (概要)
- ソースリンク

## レスポンシブ

| ブレークポイント | レイアウト |
|---|---|
| mobile (default) | 1カラム |
| sm (640px) | 2カラム (物件カード) |
| lg (1024px) | 3カラム or サイドバー付き |

## アクセシビリティ

- セマンティックHTML (main, section, nav, article)
- 適切な alt テキスト
- フォーカス可能な操作要素
- 十分なコントラスト比
